# smart-study-planner
A dynamic study planning system 
Projekt tworzony przeze mnie - pierwszy krok
## Project goal
The goal of this project is to create a simple system that helps users plan thier study time efficiently by breaking down learning materialinto manageable daily tasks.
## Features (initial version)
- adding subjects or topics 
- defining the amount of material to study
- distributing study work across multiple days
- basic prioritization 
## Technologies
- Python
- Git / GitHub 
## Status 
This project is under active development and is being built step by step as a learning-focused CS project.
## How to run
1. Clone the repository
2. Run "python main.py"
3. Follow the instructions in the terminal
